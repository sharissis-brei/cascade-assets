Cascade Assets

It is hard to manage assets inside of cascade.

Instead, organize styles, scripts, and images here, then upload the compressed, production-ready assets to cascade. The asset source files are located in the src/ folder. These assets get compiled into dist after running rake build:production or rake build:staging depending on which server you wish to build for.

This project uses the Jekyll framework to serve a mini-local version of our cascade html templates. It also utilizes the jekyll-assets plugin to give us an asset pipeline similar to Ruby on Rails.

Random Page Generator

This project has a special feature that will generate random page configurations based on the widget samples in the _includes/ folder. By running the command rake randomize, 10 pages with random layouts, widgets, titles, and content will be generated. Careful when running that command though, as it will replace any existing random pages with new ones.

This feature is useful when testing out new styles. It will allow us to make sure we don't break the layout of pages with very different configurations.

Visit /random to see the sample pages and their randomly generated configurations.
